# Shamil.dev 🚀
### Premium Full-Stack Portfolio | Synthwave Engineering Aesthetic

Welcome to the official repository of my personal portfolio. This is a high-performance, visually stunning web application built to showcase my journey as a Full-Stack Developer specializing in React, Next.js, and Mobile App Development.

![Portfolio Banner](https://images.unsplash.com/photo-1614850523296-d8c1af93d400?q=80&w=2070&auto=format&fit=crop)

## ✨ Key Features

- **🚀 Bleeding-Edge Stack**: Built with Next.js 15+ (App Router) and Tailwind CSS 4 for unparalleled performance and developer experience.
- **🌌 Synthwave Design System**: A custom-built UI featuring glassmorphism, electric glows, and a midnight-purple aesthetic.
- **🎭 Smooth Interactions**: Leverages Framer Motion for micro-animations and Lenis for high-end inertial scrolling.
- **📱 Responsive Excellence**: Fully optimized for every screen size, from mobile devices to ultra-wide monitors.
- **📂 Dynamic Project Showcase**: Interactive filtering and detailed modals for exploring my real-world, freelance, and community projects.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router, Server Components)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) (Next-gen utility-first framework)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [AutoAnimate](https://auto-animate.formkit.com/)
- **Scrolling**: [Lenis](https://lenis.darkroom.engineering/) (Smooth Inertial Scroll)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: [Inter](https://rsms.me/inter/) & [Outfit](https://fonts.google.com/specimen/Outfit)

## 🚀 Getting Started

### Prerequisites

- Node.js 20.x or later
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/muhammedshamil8/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/app`: Next.js App Router (Layouts, Pages, Global Styles).
- `src/components`: UI components (Portfolio core, Modals, Navigation).
- `src/data`: Centralized data store for projects and experience.
- `src/hooks`: Custom React hooks for UI logic and state management.
- `public`: Optimized assets and icons.

## 🎨 Design System

The portfolio uses a custom **Synthwave Engineering** theme:
- **Primary**: Electric Pink (`#ff0077`)
- **Secondary**: Warning Yellow (`#facc15`)
- **Accent**: Cyber Green (`#22c55e`)
- **Background**: Midnight Purple (`#0d0d15`)
- **Effects**: Glassmorphism (8px blur), Neon Glows, and Cyber Grid backgrounds.
